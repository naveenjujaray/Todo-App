### Deployment
In-order to work with this repo you will need to install Node.js on your machine.
```
# Clone this repository
$ git clone https://github.com/naveenjujaray/Todo-App.git

# Go into the repository
$ cd Todo-App

# Get Packages
$ npm install

# Launch/View
$ npm start
```
